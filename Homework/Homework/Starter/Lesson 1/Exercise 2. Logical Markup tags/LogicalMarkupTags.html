Logical Markup tags

Тег <abbr> позволяет описать аббревиатуру ASAP в подсказке.

Тег <cite> хорошо подходит для сносок https://itvdn.com.

Тег <code> подходит для иллюстрации кода var users = new List<User>().

Тег <strong> ставит логическое ударение для поисковых систем.

Тег <em> задает тексту курсивное начертание.

Тег <del> выделяет удаленный текст из новой версии документа.

Тег <ins> вместо удаленного текста выделяет текст из новой версии документа.

Тег <mark> помечает текст как выделенный.

Тег <q> помечает текст как цитату.

Тег <samp> отображает являющийся результатом работы программы текст. Например Build succeeded.

Тег <var> используется для выделения переменных в программе. Например, той самой users, которую мы объявили выше.
